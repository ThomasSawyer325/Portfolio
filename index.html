Barcode Scanner

Project Type: Personal

USDA FoodData Central: Nutrition Information Database

CoachAL

Project Type: Capstone

CoachAL is a fitness motivator app that connects to bluetooth sensors to gather ride data from stationary exercise bikes. It runs in the background, and as long as the rider is pedaling with their RPM thresholds, they can use the device normally. This was a group project, where I was primarily focused on Database and Bluetooth integration.

Tech Stack:

Android Studio: Android App Development
GitHub: Version Control
Wahoo Pod Cadence Sensor: Data Collection
iConsole Bluetooth Sensor: Data Collection
Heroku Postgres: Data Storage

Remote Home Server Control

Project Type: Personal

This project was created as a way for me to turn on a server from a remote location. I remote into a raspberry pi to run a python script that turns the servo motor which activates the power button on the server. Once the server is on, I can control it via TeamViewer. This allows me to turn off the server so it isn't contantly drawing power.

Technology Used:

Raspberry Pi: Remote connection via SSH
Servo Motor: Used to physically press the power button
TeamViewer: Used to control the server once it is on
